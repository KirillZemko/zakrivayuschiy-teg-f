# 💾 Проектная работа №4 — Интерактивная страница

## 📌 Описание проекта

Учебный проект, выполненный в рамках курса Яндекс Практикума.  
Страница представляет собой интерактивный интерфейс с карточками, анимациями и модальным окном.

Основное внимание уделено:
- адаптивной вёрстке
- работе с SVG
- CSS-анимациям и переходам
- работе с `<dialog>`

---

## 🚀 Функциональность

- ❤️ Кнопка лайка с анимацией
- 💾 Кнопка "Сохранить на память"
- 🪟 Модальное окно (`dialog`)
- ✨ Анимации (hover / active / focus)
- 📱 Адаптивная вёрстка (clamp, media queries)

---

## 🛠️ Технологии

- HTML5
- CSS3
  - Flexbox
  - CSS-переменные
  - clamp()
  - transition / animation
- SVG (спрайты и `<use>`)
- JavaScript (работа с dialog)
